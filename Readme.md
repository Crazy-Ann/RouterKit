# Jet Route
美柚路由库；Android平台页面路由框架

类似 ARoute 功能；

### 功能：
- Apt实现自动路由注册，支持多Module
- 路由表维护
- Activity转发 和 Action转发（支持URI页面跳转和方法调用）
